# **Expense App**
The Expense App is a Java-based application that allows users to track and manage their expenses. It utilizes Firebase as the database for storing expense data. This readme file provides an overview of the application, its features, and instructions for setting it up and running.

# Features
The Expense App offers the following features:

### 1> User Registration and Login: 
Users can create an account and log in using their credentials. <br />

### 2> Expense Management: Users can add, edit, and delete expenses.
Each expense entry includes details such as the expense amount, category, date, and description.<br />
### 3> Expense Filtering:
Users can filter expenses based on categories and dates to view specific sets of expenses.<br />
### 4> Expense Summary: 
The app provides a summary of expenses, including total expenses and expenses by category, for a given date range.<br />
### 5> User Profile:
Users can view and update their profile information, including name and email address.<br /><br />
